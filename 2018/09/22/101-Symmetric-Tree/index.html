101. Symmetric Tree

Given a binary tree, check whether it is a mirror of itself (ie, symmetric around its center).

For example, this binary tree [1,2,2,3,4,4,3] is symmetric:

But the following [1,2,2,null,3,null,3] is not:

Note:
Bonus points if you could solve it both recursively and iteratively.

Solutions:

Eventhough this is an easy problem, there are many knowledge points to cover:

  • Tree traversal
  • BFS
  • DFS
  • recursion

These also the 4 ways to solve this problem.

1. Tree traversal

Traverse the tree pre-orderly and post-orderly, comparing if they are same.

Time complexity: O(n)
Space complexity: O(n)

2. DFS(iteratively)

Here I use two stacks to implement a iteratively DFS, one for left, one for right.

Time complexity: O(n)
Space complexity: O(n)

3. BFS(iteratively)

The code is almost same, just change stack to queue.

4. Recursion

Time complexity: O(n)
Space complexity: O(1)
